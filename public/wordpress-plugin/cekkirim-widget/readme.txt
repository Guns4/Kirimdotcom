== Description ==

**CekKirim Widget** adalah plugin sederhana namun powerful yang memungkinkan Anda menampilkan form pelacakan resi (Tracking) langsung di halaman website WordPress atau WooCommerce Anda.

Sangat cocok untuk **Seller Online** yang ingin memudahkan pembeli melacak status pesanan tanpa harus meninggalkan website toko.

### Fitur Utama:
*   ✅ **Mendukung 60+ Ekspedisi**: JNE, J&T, SiCepat, AnterAja, Shopee Express, ID Express, Ninja, POS, TIKI, dll.
*   ✅ **Auto-Detect Kurir**: Pembeli cukup memasukkan nomor resi, sistem kami otomatis mendeteksi kurirnya.
*   ✅ **Desain Modern & Responsif**: Tampilan widget menyesuaikan lebar layar (Mobile Friendly).
*   ✅ **Ringan**: Menggunakan teknologi Iframe + Edge Computing dari CekKirim.com, tidak membebani server hosting Anda.
*   ✅ **Gratis Selamanya**: Tidak ada biaya langganan.

### Cara Penggunaan:
Cukup gunakan shortcode berikut di Post, Page, atau Widget Area:

`[cekkirim_widget]`

Anda juga bisa menyesuaikan warna tema agar match dengan website Anda:

`[cekkirim_widget color="red"]`
`[cekkirim_widget color="green"]`

== Installation ==

1.  Download file ZIP plugin ini.
2.  Masuk ke Dashboard WordPress -> Plugins -> Add New.
3.  Klik "Upload Plugin" dan pilih file ZIP tadi.
4.  Aktifkan plugin "CekKirim - Widget Cek Resi".
5.  Tempel shortcode `[cekkirim_widget]` di halaman yang diinginkan (misal: buat halaman baru "Lacak Paket").
